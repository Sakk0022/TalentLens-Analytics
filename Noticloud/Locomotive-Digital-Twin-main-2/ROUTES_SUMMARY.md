## 🧮 Технические детали

### Формула Haversine (для расчета расстояния)
```
R = 6371 км (радиус Земли)
Δlat = lat2 - lat1
Δlon = lon2 - lon1

a = sin²(Δlat/2) + cos(lat1) × cos(lat2) × sin²(Δlon/2)
c = 2 × arcsin(√a)
distance = R × c
```

### Интерполяция позиции
```
progress = 0.0...1.0 (текущий прогресс по маршруту)
target_distance = total_route_distance × progress

// Найти сегмент, который содержит target_distance
// Интерполировать между двумя точками сегмента

lat = lat1 + (lat2 - lat1) × segment_progress
lon = lon1 + (lon2 - lon1) × segment_progress
```

### Расчет distance per message
```
distance_per_message = (speed_kmh × interval_sec) / 3600
progress_delta = distance_per_message / total_route_distance
```
