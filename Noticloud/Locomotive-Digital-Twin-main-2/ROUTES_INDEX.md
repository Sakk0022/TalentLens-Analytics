## 🖼️ ОТВЕТ НА ВОШ ВОПРОС: "Где рисуется пунктир?"

### 📍 **Файл:** `frontend/src/components/shared/LocoMap.tsx` (строки 57-70)

```tsx
// Рисование маршрутов с пунктиром
const visibleIds = singleId ? [singleId] : Object.keys(ROUTES)
visibleIds.forEach(id => {
  const route = ROUTES[id]
  if (!route) return
  const color = LOCO_COLORS[id] ?? '#888'
  const line = L.polyline(route as [number, number][], {
    color,
    weight: 2.5,
    opacity: 0.55,
    dashArray: '8 6',  // ← 👈 ВОТ ЗДЕСь ПУНКТИР!
  }).addTo(map)
  routeLinesRef.current.set(id, line)
})
```

### Что означает каждый параметр:
| Параметр | Значение | Назначение |
|----------|----------|-----------|
| `dashArray` | `'8 6'` | Пунктир: 8px тире + 6px пропуск |
| `weight` | `2.5` | Толщина линии в пиксельях |
| `opacity` | `0.55` | Прозрачность 55% |
| `color` | Варьируется | Зелёный, жёлтый, синий, фиолетовый |

## 📞 Ответы на типичные вопросы

**Q: Почему пунктирный путь не видно?**
A: Проверьте, что:
1. Frontend запущен (`npm run dev`)
2. Маршруты загружены (откройте DevTools → console)
3. Параметр `dashArray: '8 6'` установлен в LocoMap.tsx

**Q: Как изменить пунктир?**
A: Отредактируйте [LocoMap.tsx](frontend/src/components/shared/LocoMap.tsx#L64):
- `dashArray: '5 5'` — густой пунктир
- `dashArray: '15 5'` — редкий пунктир
- `dashArray: null` — сплошная линия

**Q: Как добавить новый маршрут?**
A: 
1. Добавьте в `backend/simulator.py`: FLEET элемент с waypoints
2. Добавьте в `frontend/src/mock/simulator.ts`: ROUTES и LOCO_COLORS
3. Убедитесь, что координаты совпадают

**Q: Почему маркер прыгает?**
A: Проверьте функцию `get_position_on_route()` в backend симуляторе
