// Рисование маршрутов пунктиром
const visibleIds = singleId ? [singleId] : Object.keys(ROUTES)
visibleIds.forEach(id => {

  const color = LOCO_COLORS[id] ?? '#888'

    weight: 2.5,           // Толщина линии

## ✅ Как это работает

### 1. Начало маршрута
```
Каждый локомотив начинает с progress = 0.0
    ↓
position = waypoints[0]  (первая точка маршрута)
    ↓
Маркер появляется РОВНО в начальной точке
```

### 2. Движение по маршруту
```
Текущая скорость: 80 км/ч
Интервал: 2 сек
    ↓
distance_per_message = (80 * 2) / 3600 = 0.044 км
    ↓
progress += 0.044 / total_route_distance
    ↓
Координаты интерполируются между waypoints
    ↓
Маркер движется ПЛАВНО и ПРЯМО по маршруту
```

### 3. Правильное направление
```
Используется формула Haversine для географического расчета
    ↓
Каждый маршрут идет точно в нужном направлении:
- LOCO-001: юго-запад ↙
- LOCO-002: север ↑
- LOCO-003: юго-восток ↘
- LOCO-004: восток →
```

### 4. На карте видны пунктирные пути
```
L.polyline() с dashArray: '8 6'
    ↓
4 пунктирных линии разных цветов
    ↓
🟢 зелёный (LOCO-001)
🟡 жёлтый (LOCO-002)
🔵 синий (LOCO-003)
🟣 фиолетовый (LOCO-004)
```

## 🎓 Ключевые технические моменты

### Формула Haversine (для расчета расстояния)
```python
R = 6371  # км
dlat = (lat2 - lat1) * π/180
dlon = (lon2 - lon1) * π/180

a = sin²(dlat/2) + cos(lat1*π/180) · cos(lat2*π/180) · sin²(dlon/2)
c = 2 · arcsin(√a)
distance = R · c
```

### Интерполяция позиции
```python
def get_position_on_route(waypoints, progress: float):
    # 1. Рассчитать расстояния между точками
    # 2. Найти целевое расстояние: target = total * progress
    # 3. Найти сегмент, который содержит это расстояние
    # 4. Интерполировать позицию внутри сегмента
    lat = lat1 + (lat2 - lat1) * segment_progress
    lon = lon1 + (lon2 - lon1) * segment_progress
    return (lat, lon)
```

### Скоростное движение
```python
distance_per_message = (speed_kmh * interval_sec) / 3600  # км
progress_delta = distance_per_message / total_route_distance
progress += progress_delta  # увеличивается каждое сообщение
```
