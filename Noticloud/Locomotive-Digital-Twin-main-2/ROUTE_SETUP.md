## 🎯 Как работает движение

### 1. **Начало маршрута** ✅
Каждый локомотив **начинается ровно с первой точки** своего маршрута (индекс waypoint = 0):
- LOCO-001: начинает с Астаны
- LOCO-002: начинает с Астаны
- LOCO-003: начинает с Қарағанды
- LOCO-004: начинает с Астаны

### 2. **Расчет расстояния** ✅
Симулятор использует **формулу Haversine** для точного расчета расстояния между точками маршрута:
```python
R = 6371  # Радиус Земли в км
dlat = радиусы(lat2 - lat1)
dlon = радиусы(lon2 - lon1)
a = sin²(dlat/2) + cos(lat1) · cos(lat2) · sin²(dlon/2)
c = 2 · arcsin(√a)
distance = R · c
```

### 3. **Скоростное движение** ✅
Локомотивы движутся на основе своей **текущей скорости**:
- Расстояние за сообщение = `скорость_км/ч × интервал_сек / 3600`
- Прогресс маршрута = расстояние / общее_расстояние_маршрута
- Координаты интерполируются линейно между опорными точками

### 4. **Направление движения** ✅
Каждый локомотив движется по своему маршруту **в правильном направлении**:
- Астана → Қарағанды: юго-запад
- Астана → Петропавловск: север
- Қарағанды → Балқаш: юг-юго-восток
- Астана → Екібастұз: восток

## ⚙️ Как работает интерполяция позиций

### **Backend** (`simulator.py` строки 287-328)
```python
def get_position_on_route(waypoints: list, progress: float) -> tuple[float, float]:
    """Получить интерполированную позицию по маршруту на основе прогресса (0.0 до 1.0)."""
    # 1. Рассчитать расстояния между каждой парой точек
    segments = []
    total_dist = 0.0
    for i in range(len(waypoints) - 1):
        dist = haversine(waypoints[i], waypoints[i + 1])
        segments.append(dist)
        total_dist += dist
    
    # 2. Найти целевое расстояние на основе progress
    target_dist = total_dist * progress
    
    # 3. Найти сегмент, который содержит целевое расстояние
    for i, seg_dist in enumerate(segments):
        if current_dist + seg_dist >= target_dist:
            # 4. Интерполировать позицию внутри этого сегмента
            segment_progress = (target_dist - current_dist) / seg_dist
            lat = waypoints[i][0] + (waypoints[i+1][0] - waypoints[i][0]) * segment_progress
            lon = waypoints[i][1] + (waypoints[i+1][1] - waypoints[i][1]) * segment_progress
            return (lat, lon)
```

**Результат:** Плавное движение по маршруту между 5 опорными точками ✅
